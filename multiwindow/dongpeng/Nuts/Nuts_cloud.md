# 介绍
  - 坚果云是上海亦存网络科技有限公司推出的一款便捷，安全的专业网盘产品, 不仅支持文件自动同步，共享，备份功能提高了办公效率。还是微软加速器云存储合作伙伴，IBM全球企业扶持的计划成员，同时还支持不同系统如：windows,
  mac,ios,linux, android, web, windows phone,web 7大系统上的使用。 本次测试的是windows.
# 测试环境
  - windows 10 + 坚果云5.1.4
