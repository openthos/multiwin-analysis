# 说明
  - openthos 云桌面版本 根据seafile版本2.2.9，拆分原有逻辑使其实现左右布局的分离
  
# 功能需求
  - 登录
    - 登录页面页面逻辑分为已添加帐号  和 未添加帐号
      - 已添加帐号
        - 已添加帐号默认直接进入主页面----> BrowserActivity
      - 未添加帐号
        - 初次登录以及没有添加帐号，在主页面进行了判断处理，没有帐号的会进入到添加帐号页面
    
    
  - 需求为：
    - 沿用原有逻辑，在添加账户页面进行布局调整，目前风格为左边对齐效果
    - 点击添加账户
